# TaskTracker
A gentle task tracker with daily tasks, weekly/monthly goals, progress sliders, and a streak counter.

## Features
- 📅 Click any date to add tasks
- 🎚️ Progress sliders (0-100%)
- ✅ Checkbox completion
- 🌱 Weekly & 🌳 monthly goals
- 🔥 Streak counter
- 💾 Auto-saves to localStorage

## Files
- `index.html` - HTML structure
- `style.css` - Styling
- `script.js` - Functionality

## How to Use
1. Place all 3 files in the same folder
2. Open `index.html` in any browser
3. Click a date, add tasks, adjust progress
4. Complete all daily tasks to grow your streak
